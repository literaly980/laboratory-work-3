Чтобы программа работала нужно скачать библиотеки из requirements, а так же установить ffmpeg, инструкция как его установить ниже:

1. Откройте https://ffmpeg.org/download.html.

На странице загрузки (например, Gyan) найдите и скачайте файл с названием, похожим на ffmpeg-release-full.zip или ffmpeg-*-essentials_build.zip.


3. Распакуйте загруженный ZIP-архив.

Найдите папку bin внутри распакованного каталога. В этой папке находятся исполняемые файлы: ffmpeg.exe, ffprobe.exe и ffplay.exe.

Переместите эту папку bin в надежное и легкодоступное место на вашем диске, например:

C:\FFmpeg (внутри этой папки должна быть папка bin)

Или сразу папку bin в корень диска: C:\bin\ffmpeg

Предположим, ваш конечный путь к исполняемому файлу выглядит так: C:\FFmpeg\bin\ffmpeg.exe.

3. Настройка переменной PATH
Для того чтобы Python и pydub могли найти ffmpeg.exe из любого места в системе, необходимо добавить путь к папке bin в системную переменную PATH.

Нажмите кнопку Windows и начните вводить "Переменные среды".

Выберите пункт "Изменение системных переменных среды" (Edit the system environment variables).

В открывшемся окне "Свойства системы" нажмите кнопку "Переменные среды..." (Environment Variables...).

В нижней части окна ("Системные переменные" / System variables) найдите переменную с именем Path и выберите ее.

Нажмите кнопку "Изменить..." (Edit...).

В новом окне нажмите "Создать" (New) и вставьте полный путь к вашей папке bin.

Пример: C:\FFmpeg\bin

Нажмите "ОК" во всех открытых окнах.
